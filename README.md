# 💊 Medicine Recommendation System

## 📌 Overview
This project is a Machine Learning-based Medicine Recommendation System that suggests appropriate medicines based on user symptoms. It helps in preliminary diagnosis and assists users in identifying possible treatments.

---

## 🚀 Features
- Symptom-based disease prediction
- Medicine recommendation system
- User-friendly interface (Flask app)
- Machine Learning model integration
- Fast and efficient predictions

---

## 🧠 Tech Stack
- Python
- Machine Learning (Scikit-learn)
- Streamlit
- Pandas, NumPy
